# winEasyX

只是一个最简单的桌面宠物的cpp style实现，使用了EasyX的IMAGE类来加载png，其余部分均使用win32 api，目前功能包括:

* [x] 循环播片
* [x] 长按拖拽
* [x] 待机动画
* [x] 托盘图标
* [x] 右击菜单
* [x] 点击动画
* [x] 左右方向
* [x] 随机行走
* [ ] 拖拽动画(缺少美术资源)
